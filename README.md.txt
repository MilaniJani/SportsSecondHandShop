### README.md

・アプリケーション名：SportsSecondHandShop

1. プロジェクト概要
 A. ユーザーが重心になって、スポーツに関連する中古品を販売するウェブサイト。
 B. ユーザーが自分のスポーツ中古品を販売登録ができる。
　また、他のユーザーが販売する商品を購入することができる。

2. プロジェクトの機能
  A.会員関連
	a-1. 会員加入
	a-2. ログイン・ログアウト
	a-3. 会員情報の閲覧・修正
  B.QnA掲示板
	b-1. 一般ユーザー：お問い合わせ
	b-2. 管理者：お問い合わせの管理
  C.買い物
	c-1. 販売したい商品を直接登録する(メインページへ反映)
	c-2. 販売している商品がメインページで見れる
	c-3. 商品の詳細な情報を見て、購入
